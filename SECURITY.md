# Security Policy

## Purpose

This repository is an educational XSS defense and web security awareness resource.

## Reporting Issues

If you find an error, unsafe wording, outdated guidance, or a security concern in this repository, please open an issue or submit a pull request.

## Responsible Use

All material in this repository is intended for legal, ethical, educational, and defensive use only.

Do not use any content from this repository against websites, applications, accounts, or systems without explicit written authorization.

## Scope

This repository does not maintain runnable attack payloads, credential theft examples, evasion chains, or exploit tooling.
